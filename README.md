# ModMouse

> A lightweight, high-performance AutoHotkey utility that enhances mouse input through configurable modifier layers, accelerated scrolling, and intelligent mouse-button chording.

---

## Overview

ModMouse is designed to extend the capabilities of a standard mouse without requiring additional drivers or vendor software.

Instead of binding features to dedicated hardware, ModMouse introduces configurable modifier layers that allow existing mouse buttons to perform multiple functions naturally and efficiently.

The project prioritizes:

- Performance
- Predictability
- Simplicity
- Configurability
- Native Windows behaviour

---

## Features

### Current

- Modifier-based scrolling
- Smooth accelerated scrolling
- Mouse-button chording
- Opera/Chromium compatibility fixes
- Single-file architecture
- Lightweight runtime overhead

### Planned

- Fully configurable key bindings
- Multiple acceleration curves
- Per-application profiles
- Debug mode
- Additional modifier layers

---

## Design Philosophy

ModMouse is intentionally built as a **single-file AutoHotkey application**.

Rather than relying on external libraries or complex project structures, every feature is contained within a single, well-organized script.

This approach keeps the project:

- Easy to understand
- Easy to modify
- Easy to distribute
- Easy to audit
